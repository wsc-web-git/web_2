Summary for Students

Learn Logic in Simple Words

Logic is the study of correct thinking. In this chapter we learned about statements, which are sentences that are either true or false, but not both. We do not count questions, orders, or exclamations as statements.

We learned about simple statements and how to join them with connectives like AND, OR, NOT, IF-THEN, and IF-AND-ONLY-IF. The word AND is true only when both parts are true. The word OR is true when at least one part is true. NOT changes true to false and false to true. IF-THEN is false only when the first part is true and the second is false.

We also saw truth tables. A truth table shows all possible true and false values for a statement. It helps us check if a compound statement is always true, always false, or sometimes true.

We studied logical equivalence, which means two statements have the same truth value in every case. We also learned about tautology, contradiction, and contingency.

Practice is very important. Make your own examples, draw truth tables slowly, and read each connective in simple English. When you understand the meaning, the symbols become easy. Keep revising daily, and logic will help you in mathematics, computer science, and in clear everyday thinking.
